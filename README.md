# SolVote 🗳️

A fully decentralized voting system built on Solana.

## 🚀 Live Demo
[https://sol-vote.netlify.app/](https://sol-vote.netlify.app/)

## 🔧 Tech Stack
- **Frontend:** React + TypeScript
- **Backend:** Rust (Anchor)
- **Blockchain:** Solana Devnet
- **Wallet:** Phantom
- **RPC:** Helius API

## ✨ Features
- Wallet authentication
- Create polls with custom timing
- Approved-voter system
- One vote per wallet
- Real-time results
- Fully on-chain data storage
